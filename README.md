# SCT_WD_003
Modern Tic Tac Toe Game
A classic Tic Tac Toe game brought to life with a modern, responsive design and smooth animations using HTML, CSS (Tailwind CSS), and JavaScript.

Clear Game Status: The UI clearly indicates whose turn it is, the winner, or if the game is a draw.

Self-Contained: All necessary HTML, CSS, and JavaScript are in a single, easy-to-manage file.
